# Messenger Web Application
Учебный проект по созданию веб-мессенджера<br><br>

## Installation

Инициализация проекта:

```bash
npm install
```

Команда для запуска проекта в режиме разработки:

```bash
npm run dev
```
Команда для сборки и запуска проекта:

```bash
npm run start
```
